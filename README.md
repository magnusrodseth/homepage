## Running the application ✅

This project uses [`npm` workspaces](https://docs.npmjs.com/cli/v7/using-npm/workspaces). Thus, you can perform all needed commands from the root `/homepage` directory.

```sh
# Navigate to the homepage directory
cd homepage

# Start client in development mode
npm run client:dev

# Start cms in development mode
npm run cms:dev

# Generate types for client using graphql-codegen
npm run client:gen
```
